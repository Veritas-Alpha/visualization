# 项目简介

我们正在构建一个能够 **自动生成美国和印度上市公司分析报告的智能分析 Agent**。现在项目已经有得到一些投资。  
用户选择目标公司后，系统会自动抓取 SEC、FMP 等数据源的原始信息，并通过 LLM 使用多种分析方法，对公司进行结构化分析，最终生成高质量的 PDF 报告。

目前报告内容已能完整覆盖分析逻辑，但 **文字信息过重、可读性一般**，急需加入更直观、更美观的自动化可视化组件，以提升专业度与阅读体验。

这个 repo 里放了一个例子：AAPL.pdf，可供参考。

---

# 工作内容

你的主要任务是为报告加入 **自动化的可视化与信息展示模块**，包括但不限于：

- 针对关键财务数据，选择适合的图表格式并自动生成（如趋势线、同比/环比柱状图、结构图等）
- 针对文字类分析内容，设计类似“PPT 信息块”的自动化视觉布局（重点摘要卡片、亮点/风险模块等）

整个流程目前已实现全自动化，因此 **可视化模块也需保持可扩展性与自动化**。
